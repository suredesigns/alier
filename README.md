## What is Alier

Alier is a JavaScript framework that comprehensively supports the development of mobile apps and web apps. Alier has the following features.

- With Alier, you can develop Android and iOS apps with a single JavaScript code. Web apps can also run with a single code if they do not use platform-dependent functions.

- In Alier, Model and View are separated by architecture. This reduces the variation in code quality caused by differences in understanding and interpretation of application architecture, making it easier for members of a team to understand each other's work.

- Alier does not require complicated installation. You can start development by simply downloading and linking the distributed library.

- You can start app development with just basic knowledge of JavaScript and HTML.

For more information about Alier, please see the [Alier official website](http://alier.dev/).

Also, we develop [Alier on Node](https://github.com/suredesigns/alier-node), which is a library to build a web server.

## How to install

Alier comes in two versions: Alier for mobile and Alier for Web.
- Alier for mobile: A framework for mobile apps
- Alier for Web: A framework for web apps

Alier has released Alier for mobile and Alier for Web together as a framework for apps. You can download it from [Alier framework for apps](https://github.com/suredesigns/alier/releases/latest). You can start development by linking the downloaded library. For detailed installation, please refer to [Installation](http://alier.dev/StartGuide/gettingstarted/).

## Current Status

The library released is the beta 2 version.

- Alier for mobile currently works without any problems. However, there is a possibility that the specifications will change in the future, and compatibility may be lost. Please be aware of this when installing.

- Alier for Web currently works without any problems. However, future changes to the specifications will almost certainly make it incompatible. Please be aware of this when installing.

- Alier on Node is at an experimental level. Please use it as a reference.

As mentioned above, Alier will undergo specification changes in the future. Therefore, if you have any requests, please contact us at [GitHub issues](https://github.com/suredesigns/alier/issues).
