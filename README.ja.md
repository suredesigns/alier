## Alierとは

Alier(アリエ) は、モバイルアプリ、Web アプリ開発を包括的にサポートする JavaScript フレームワークです。Alier は以下のような特徴があります。

- Alier を使えば Android、iOS のアプリをJavaScriptの単一コードで開発することが出来ます。Web アプリもプラットフォーム依存の機能を使用しなければ単一コードでの動作が可能です。

- Alier では Model と View の枠組みが最初から決まっています。それによってアプリケーションアーキテクチャに対する理解度の差や解釈の違いによって生まれるコード品質のバラつきを低減することが出来、チーム内ではお互いの作業内容の把握が容易になります。

- Alier に複雑な導入はありません。配布しているライブラリをダウンロードしてリンクするだけで開発を始められます。

- Alier はJavaScriptとHTMLの基礎知識だけあればアプリ開発を始められます。

Alier について詳しくは[Alier公式サイト](http://alier.dev/)を確認してください。

また、関連プロダクトとして [Alier on Node](https://github.com/suredesigns/alier-node) があります。[Alier on Node](https://github.com/suredesigns/alier-node) はサーバ構築のためのライブラリです。

## インストール方法

Alierには Alier for mobile と Alier for Web があります。
- Alier for mobile : モバイルアプリ向けのフレームワーク
- Alier for Web : Webアプリ向けのフレームワーク

Alierではこの Alier for mobile と Alier for Web をまとめてアプリ向けフレームワークとして公開しています。[Alier アプリ向けフレームワーク](https://github.com/suredesigns/alier/releases/latest)からダウンロードできます。ダウンロードしたライブラリをリンクすることで開発を始められます。詳しいインストール方法については[公式サイトのインストール方法](http://alier.dev/StartGuide/gettingstarted/)を確認してください。

## 現在のステータス

公開中のライブラリはβ2版です。

- Alier for mobile については現在は問題なく動作する状態です。ですが、今後仕様変更が入る可能性があるので、互換性がなくなる可能性があります。導入する方はその点ご留意ください。

- Alier for Web については現在は問題なく動作する状態です。ですが、今後の仕様変更でほぼ確実に互換性がなくなります。導入する方はその点ご留意ください。

- Alier on Node は実験的なレベルです。参考程度に見ていただけると幸いです。

上記のように、今後Alierには仕様変更が入ります。そのため、このような仕様にしてほしい等の要望があれば、こちらの [GitHub issues](https://github.com/suredesigns/alier/issues) にご連絡いただければ幸いです。
